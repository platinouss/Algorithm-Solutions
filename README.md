### :exclamation: 규칙
- 하루에 한 문제 이상 해결한다. (습관화되면 개수 올릴 예정)
- **불가피한 일**이 잡혔을 경우 그 날을 대비해 미리 풀어놓을 수 있다. (해당 날짜를 커밋 메시지에 남겨 놓기)

### :exclamation: 벌칙
- 정해진 수 만큼의 문제를 해결하지 않았다면, 밀린 문제 수의 2배를 해결해야 한다.
    - 만약 15일, 16일 연속으로 풀지않았을 경우에는 17일에 (1+1)*2 = 4문제를 해결해야 한다.
